Frontend Tech Stack

You said no Streamlit. What should we build with?

A

Flask + Vanilla HTML/CSS/JS (Recommended)

Lightweight Python backend (Flask) serving a single-page app. No build tools, no npm, no Node. Just Python + HTML/CSS/JS files. Easy to run: python app.py. Fastest to build and deploy.

B

FastAPI + Vanilla HTML/CSS/JS

Same as A but with FastAPI instead of Flask. Async-native, auto-generated API docs at /docs. Slightly more modern. Also just python app.py.

C

Next.js / React

Full React frontend with separate Python API backend. Requires Node.js, npm, build step. More complex setup but more scalable for future features.
